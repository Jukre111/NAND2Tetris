Zusätzlich zu Python werden die Packete: sys, os, nltk benötigt sodass diese vor dem ausführen sinnvollerweise vorhanden sein sollten.
Bei nltk ist es nötig zusätzlich "punkt" zu downloaden, indem man über die Kommandozeile am besten einfach folgendes macht:

$ python
>>> import nltk
>>> nltk.download('punkt')
